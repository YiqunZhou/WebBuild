Shadow ¬∑ Touch

Interactive Python Program, 2023

I used python hand tracking library MediaPipe to build an interactive virtual touching game, where you can see the shadow of your hands being fragmented by tree shadows, touched by the box, or when you grasp the invisible ball in the screen you can hear sounds.

This project is part of the Game Jam event organized by Icosa Magazine. After the event I also edited three pages to be put in the final book.
